🚀 Documentación Completa y Detallada del Proyecto: Portafolio Web Personal
¡Bienvenido a mi repositorio de portafolio web personal! Este proyecto fue conceptualizado, diseñado y desarrollado completamente desde cero con el objetivo principal de construir una carta de presentación profesional, moderna y altamente funcional que refleje mis capacidades, habilidades técnicas y los proyectos en los que he estado trabajando como desarrollador.

📌 ¿De qué trata mi proyecto?
Básicamente, quise crear mi propia página web personal para presentarme formalmente como desarrollador de software. Es un espacio digital interactivo donde cualquier reclutador, colega o persona interesada puede entrar, conocerme a fondo y ver de manera directa tanto lo que sé hacer como los proyectos que he desarrollado.

El propósito central es romper con los currículums planos tradicionales, ofreciendo una experiencia visual fluida que demuestre desde el primer momento mis competencias técnicas y mi atención al detalle en el diseño y la experiencia de usuario (UX/UI).

🛠️ ¿Qué tecnologías utilicé y por qué?
Para la construcción de este portafolio, tomé la decisión consciente de programarlo todo utilizando tecnologías web nativas (Vanilla Code), evitando depender de frameworks o librerías pesadas. Quería demostrar que domino las bases fundamentales del desarrollo web de forma sólida y limpia:

HTML5: Lo utilicé como la columna vertebral de la aplicación para armar toda la estructura semántica y el contenido de la página de forma ordenada, accesible y optimizada.

CSS3: Me sirvió para diseñar toda la capa visual, crear una interfaz moderna basada en temas oscuros profesionales y asegurar que la plataforma sea completamente responsiva, adaptándose a la perfección tanto a pantallas grandes de computadoras como a dispositivos móviles.

JavaScript (Vanilla ES6+): Lo implementé para dotar de vida a todo el sitio, controlando toda la lógica de interactividad, animaciones dinámicas y gestión de vistas sin intermediarios.

⚡ ¿Qué características y funcionalidades principales tiene mi portafolio?
Para marcar la diferencia y ofrecer una navegación atractiva, le programé diversos componentes interactivos avanzados:

El menú superior dinámico (Header interactivo): Hice que la barra de navegación superior detecte automáticamente el movimiento del usuario, cambiando de tamaño y aplicando efectos visuales al hacer scroll hacia abajo. Además, incluye la funcionalidad de Scroll Spy, lo que permite que el menú ilumine de manera inteligente la sección exacta en la que se encuentra el visitante.

Gráficas de habilidades animadas (Skill Bars/Circles): Diseñé elementos visuales con contadores numéricos y barras dinámicas que comienzan a avanzar y subir de forma fluida desde cero hasta el porcentaje real de dominio solo en el instante exacto en que el usuario baja la pantalla hasta esa sección.

Efecto visual dinámico en tarjetas (Spotlight Effect): A las tarjetas que muestran mis proyectos les programé un comportamiento interactivo en JavaScript para que reaccionen sutilmente al movimiento del cursor, generando un brillo direccional que sigue el puntero del mouse y le otorga un acabado sumamente profesional y pulido.

Página de detalles complementarios (detalles.html): Estructuré una vista secundaria modular gestionada de forma dinámica para permitir profundizar en especificaciones técnicas de manera organizada.

Optimización y Despliegue: Todo el código fuente está estructurado bajo una arquitectura limpia y modular, optimizado para garantizar tiempos de carga casi instantáneos y preparado para subirse y mantenerse en funcionamiento de forma gratuita mediante plataformas como GitHub Pages.
